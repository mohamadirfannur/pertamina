# PLTS Site Intelligence - Pertamina NRE

Dashboard untuk monitoring dan analisis data PLTS (Pembangkit Listrik Tenaga Surya) Pertamina New & Renewable Energy.

## Fitur

- Dashboard real-time monitoring PLTS
- Visualisasi data produksi energi
- Analisis performa site
- Interface modern dengan dark mode
- Responsive design

## Cara Menjalankan

### Metode 1: Buka langsung di browser
Cukup double-click file `pertamina-plts-dashboard.html` atau buka dengan browser favorit Anda.

### Metode 2: Menggunakan local web server

**Dengan Python:**
```bash
python3 -m http.server 8000
```
Kemudian buka `http://localhost:8000/pertamina-plts-dashboard.html`

**Dengan Node.js:**
```bash
npx http-server -p 8000
```
Kemudian buka `http://localhost:8000/pertamina-plts-dashboard.html`

## Teknologi

- HTML5
- CSS3 (dengan custom properties)
- JavaScript (vanilla)
- Plus Jakarta Sans font
